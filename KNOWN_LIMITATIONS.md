# Known Limitations

## Stable enough to use

- local-first 2D planning
- room furnishing
- active-floor 3D viewing
- local export workflows

## Still fragile

- the runtime still uses a compatibility bridge for browser-global modules
- state is not yet fully centralized
- cloud sync is experimental and not collaborative-safe
- some UI surfaces still rely on inline handlers and string-built markup
- the 3D runtime still depends on CDN-hosted libraries

## What this means

This app is useful, but it is still being structurally hardened. Treat it as a valuable codebase under repair, not a finished platform.
